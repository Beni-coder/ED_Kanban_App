# Scripts - Kanban Studio

Start and stop scripts for the Docker-based application.

## Files
- `start.bat` / `start.sh` - Build and start the Docker container (detached)
- `stop.bat` / `stop.sh` - Stop and remove the Docker container

## Usage
- Windows: `scripts\start.bat` and `scripts\stop.bat`
- Mac/Linux: `scripts/start.sh` and `scripts/stop.sh`

After starting, the app is available at http://localhost:8000
